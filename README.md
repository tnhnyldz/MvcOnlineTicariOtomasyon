
# MVC Online Ticari Otomasyon 

This project was developed in 2 months.
Inspired by Murat Yücedağ MVC Online Ticari Otomasyon Course



## Used Technologies



- Microsoft .NET Technologies 
- Microsoft SQL Server
- JQuery 
- Entity Framework
- Bootstrap 5

- Code First

- Google Charts 
- Triggers
